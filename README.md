## EventTrackerProject

### Summary / Overview

### REST API Endpoints
* [VERB][ROUTE]
|GET| /api/outbreaks | Retrieve all outbreaks|
|GET| /api/outbreaks/{id} | Retrieve outbreaks |
|POST| /api/outbreaks | Post a new break|
|PUT|/api/outbreaks/{id} | Replace an outbreak  |
|DELETE| /api/outbreaks/{id}| Delete an outbreak|

### Technologies used
* Spring Rest API
* JPA/Hibernate/JPQL
* MySQL
